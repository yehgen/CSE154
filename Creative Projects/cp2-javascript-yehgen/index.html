Top secret coding related activities I have done

I went into DubHacks with both the introductory Computer Programming courses for Java (CSE142 and CSE143) under my belt and a little bit of self taught HTML, CSS, and JavaScript. Using the React framework for JavaScript, we designed a web application intended to help users build habits based on the notion that it takes approximately 66 days of consistently performing a task to transform the given task into a habit. We learned React as we went along, with minimal prior knowledge.
